Expense Tracker (Python)

Project Overview  
This project is a simple command-line expense tracker built using Python. It allows users to add expenses, view all expenses, and calculate the total spending. The data is stored in a text file for persistence.

Features  
- Add new expenses with name and amount  
- View all saved expenses  
- Calculate total expenses  
- Stores data in a file for future use  

Technologies Used  
- Python  
- File Handling  

Project Structure  

expense-tracker/  
│  
├── expense.py  
├── expenses.txt  
└── README.md  

How to Run  

1. Run the program using Python:  
python expense.py  

2. Choose options from the menu:  
- Add Expense  
- View Expenses  
- Exit
